# 🤖 Mission 2 (Grand Saga) Quiz Bot

This bot delivers premium dark-fantasy vocabulary quizzes for **Mission 2 (Grand Saga)**.

### 🛠️ Tech Stack:
- **Backend:** Python (Flask + Threading)
- **Database:** MongoDB Atlas (Persistent state & analytics)
- **Deployment:** Hugging Face Spaces
- **Platform:** Telegram

### 🚀 Commands:
- `/quiz [count] [timer]` - Start a session (e.g., `/quiz 20 15`)
- `/stats` - View the Mission 2 Leaderboard
- `/stop` - Stop the current session
